Requisitos funcionais (RF'S):                                |  Descrição:
                                                             |
- Cálculo de média                                           |  O sistema deve fazer o cálculo das médias, somando as notas e dividindo-as pela quantidade de notas.
- Identificação de status dos alunos de acordo com a nota    |  O sistema deve identificar de acordo com a média se o aluno está abaixo, acima ou se tem a                                                                         |  melhor média
- Inserção de dados                                          |  O sistema deve capturar os dados inseridos pelo aluno para realizar as operações
===============================================================================================================================================
Requisitos não-funcionais (RNF'S):                           | Descrição:
- Clareza nas mensagens                                      | O sistema retornará mensagens claras para que o usuário possa navegar sem dúvidas ou dificuldades
- Organização de tarefas                                     | O sistema será dividido por partes: processamento (recepção de dados e cálculos) e respostas 
===============================================================================================================================================
Regras de negócio (RN'S):                                    | Descrição:

- Identificação de alunos                                    | O aluno deve se identificar com nome, ano e turma
- Gerar arquivo .txt de resultados                           | O sistema deve emitir um arquivo ".txt" a fim de retornar todos os resultados
- Resultados de média                                        | Se a média de um aluno for menor que 7, será reprovado; Caso contrário, será aprovado
- Validação de dados                                         | O sistema não deve continuar com as operações se existirem dados incompletos ou inválidos
